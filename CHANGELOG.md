# CHANGELOG

## [1.0.0] — 2026-07-16

### 规则引擎 (`src/core/`)

- 完整掼蛋牌型系统：单张、对子、三带二、顺子（≥5）、木板（连续对子≥3）、钢板（连续三带二≥2）、炸弹（4+）、同花顺、天王炸
- 逢人配（红心级牌可替代任意牌组成牌型）
- 出牌合法性校验：牌型匹配 + 大小比较
- 随机发牌 + 进贡/还贡/抗贡完整流程
- 升级系统：头游/二游/三游/末游 → 1-3 级升级，双下翻倍
- 过 A 获胜判定

### AI 三档 (`src/ai/`)

- **Easy**（弱档）：贪最小出牌、不炸、无视队友
- **Normal**（中档）：让队友、留炸、跟最小非炸、保留结构
- **Hard**（强档）：确定化蒙特卡洛 (DMC) — 记牌推算 × 多副模拟到终局 × 选平均得分最高的着法
  - 喂牌：队友快出完自动出小牌让队友跑
  - 控场：对手临近获胜出不可压牌
  - 护队友：炸弹保牌权
  - Baseline guard：以启发式 hard 为下限防止 MC 出烂牌
  - 多次自对弈调优：有非炸不浪费炸弹 (-83%)、便宜顶队友仅在有威胁时

### 3D 国风渲染 (`src/render/`)

- Three.js 牌桌场景 + 四向布局（你/下家/队友/上家）
- CSS 卡片降级方案（靠相机距离切换 3D ↔ CSS）
- 国风墨体材质纹理
- 理牌动画、出牌动画、结算动画
- 炸弹特效（按牌数分级：震动 + 粒子 + 屏幕效果）
- 逢人配高亮标记
